# EDUCARE – AI Based Mathematics Learning Support System

## Project Description
EDUCARE is an AI-based mathematics learning support system designed for Ethiopian secondary school students (Grade 9–12). The system identifies student learning gaps and generates personalized practice materials.

## Objective
The main objective is to:
- Identify weak mathematics topics
- Provide personalized exercises
- Help teachers monitor student progress
- Support mastery based learning
