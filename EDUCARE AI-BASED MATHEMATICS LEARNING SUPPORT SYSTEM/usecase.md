# EDUCARE Use Cases

## Student Use Cases

### Take Quiz
**Actor:** Student  
**Steps:**  
1. Open quiz  
2. Answer questions  
3. Submit quiz  
4. System calculates score  

### View Results
**Actor:** Student  
**Steps:**  
1. Open results  
2. Select quiz  
3. View score  
4. View weak topics  

### Practice Activities
**Actor:** Student  
**Steps:**  
1. Open practice list  
2. Select exercise  
3. Complete practice  

---

## Teacher Use Cases

### View Student Progress
**Actor:** Teacher  
**Steps:**  
1. Login  
2. Open dashboard  
3. Select student  
4. View performance  

### Approve Materials
**Actor:** Teacher  
**Steps:**  
1. Open material list  
2. Review AI-generated content  
3. Approve or reject  

### View Skill Gaps
**Actor:** Teacher  
**Steps:**  
1. Open dashboard  
2. Select class  
3. View gap analysis  

---

## Admin Use Cases

### Manage System & Data
**Actor:** Admin  
**Steps:**  
1. Login  
2. Configure system settings  
3. Backup database  
4. Maintain AI data  

**Note:** Admin does **not** manually add student or family accounts; users register themselves through the system.  

---

## Family Use Cases

### Link Account to Child
**Actor:** Family (Parent/Guardian)  
**Steps:**  
1. Login to EDUCARE  
2. Enter child-generated unique code  
3. System links parent account to child  

### View Report
**Actor:** Family (Parent/Guardian)  
**Steps:**  
1. Login  
2. Access linked child’s dashboard  
3. View progress, weak areas, and AI recommendations
